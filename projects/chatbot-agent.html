🤖 AI Chatbot Agent (LangChain)

Статус: Завершено | Стек: Python, LangChain, OpenAI API, Docker

💡 Обзор проекта

Мультишаговый чатбот, разработанный для анализа документов (PDF/DOCX) и ответов на вопросы пользователя по их содержанию. Система использует архитектуру агента LangChain, что позволяет ей выполнять сложные цепочки действий — от загрузки данных до обращения к внешним API.

Ключевые особенности:

  • Retrieval Augmented Generation (RAG): Обеспечивает точные ответы, основанные на предоставленной документации.
  • Multi-step Agents: Автоматическое планирование действий для решения сложной задачи.
  • Контейнеризация: Полностью упакован в Docker для упрощенного развертывания.

🖼️ Демонстрация

Здесь будет размещена интерактивная демонстрация или скриншот работы бота.

⚙️ Как запустить проект?

  1. Клонировать репозиторий проекта в папку `projects/chatbot-agent.
  2. Создать файл `docker-compose.yml` и настроить переменные окружения (OPENAI_API_KEY).
  3. Запустить командой: docker compose up -d
